# Commanding the Console 

## Tips and Tricks

- Making Commits:
    -Commit messages should be short and meaningful
    -Make frequent commits
- VS Code Keyboard Shortcuts
    - **ctrl** + **`** for toggleing the terminal window
    - **ctrl** + **k** followed by **s** to save all
